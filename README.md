# 🌐 Language Translator (English to Tamil)

A simple Python-based **language translation tool** using the `googletrans` library. This project translates English text into Tamil instantly, demonstrating basic Natural Language Processing (NLP) capabilities.

## 🚀 Features

* 🔤 Translate English text to Tamil
* ⚡ Fast and lightweight implementation
* 🧠 Uses Google Translate API via `googletrans`
* 🐍 Beginner-friendly Python project

## 🛠️ Tech Stack

* **Language:** Python
* **Library:** `googletrans`

## 📌 How It Works

1. Input text is defined in the program
2. `Translator()` object is created
3. Text is translated from English (`en`) to Tamil (`ta`)
4. Translated output is printed

### 1️⃣ Install Dependency

```bash
pip install googletrans==4.0.0-rc1
```

### 2️⃣ Run the Script

```bash
python app.py
```
